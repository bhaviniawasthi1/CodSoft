# CodSoft Java Internship Tasks

This repository contains all 5 tasks completed during my Java Programming Internship at CodSoft. The projects focus on core Java concepts including control flow, object-oriented programming, file handling, and basic API integration.

---

## Tasks Overview

### Task 1: Number Guessing Game (`codsoft_task1.java`)

A console-based game where the user guesses a randomly generated number.

**Features:**

* Random number generation (1‚Äì100)
* Limited number of attempts
* Hints (Too High / Too Low)
* Multiple rounds support
* Score tracking

---

### Task 2: Student Grade Calculator (`codsoft_task2.java`)

Calculates total marks, average percentage, and assigns grades.

**Features:**

* Accepts marks for multiple subjects
* Calculates total and percentage
* Grade assignment logic
* Input validation

---

### Task 3: ATM Interface (`codsoft_task3.java`)

Simulates basic ATM operations using object-oriented design.

**Features:**

* Check balance
* Deposit and withdraw money
* Input validation
* Separate `BankAccount` class

---

### Task 4: Currency Converter (`codsoft_task4.java`)

Converts currency using real-time exchange rates.

**Features:**

* Base and target currency selection
* Live exchange rate API integration
* Conversion calculation
* Error handling

---

### Task 5: Student Management System (`codsoft_task5.java`)

A system to manage student records with persistent storage.

**Features:**

* Add, remove, search, and edit student details
* Display all students
* File handling using serialization
* Input validation

---

## Technologies Used

* Java (Core Java)
* Object-Oriented Programming (OOP)
* File Handling (Serialization)
* HTTP API Integration
* Console-based Applications

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/codsoft.git
```

2. Navigate to the folder:

```bash
cd codsoft
```

3. Compile any task:

```bash
javac codsoft_taskX.java
```

4. Run the program:

```bash
java codsoft_taskX
```

(Replace `X` with task number 1‚Äì5)
